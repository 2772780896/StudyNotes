## 创建路径

```python
file_path = r'C:\Users\...\abc11.py'
```

使用反斜杠 `\` 表示路径时，`r''` 原始字符串声明 `\` 不进行转义。

## 打开文件

```python
open(file, mode='r', buffering=None, encoding=None, errors=None, newline=None, closefd=True)
```

### mode 参数

| 模式 | 说明 |
|------|------|
| `'r'` | 以文本模式读取（默认） |
| `'w'` | 写入，文件存在则截断（覆盖），不存在则创建 |
| `'x'` | 创建并写入，文件已存在则抛出 `FileExistsError` |
| `'a'` | 追加，写入内容追加到文件末尾 |
| `'b'` | 二进制模式 |
| `'t'` | 文本模式（默认） |
| `'+'` | 读写模式（同时支持读取和写入） |
| `'U'` | 通用换行模式（已弃用） |

## 读取文件

```python
opened_file = open(file_path, mode='r', encoding='utf-8')
text1 = opened_file.read()
```

- `read()` 可传入字符数参数，无参数则全部读取
- 文件有一个描述符指针，读取后指针后移，下次读取从新位置开始

## 文件描述符

```python
opened_file.tell()    # 查看当前描述符位置
opened_file.seek(1)   # 移动描述符到指定位置
```

## 写入文件

### 写入缓冲区

```python
opened_file1 = open(r'./wrote_file', mode='w', encoding='utf-8')
opened_file1.write('abc')
```

数据先写入缓冲区，之后才实际写入磁盘。

### 冲洗缓冲区

缓冲区数据在以下情况下被写入磁盘：
- 手动调用 `opened_file1.flush()`
- 文件被关闭时自动冲洗
- 缓冲区满时自动冲洗

## 关闭文件

```python
open(file_path).close()
```

不关闭文件会持续占用内存。

## with 语句

```python
with open(file_path, encoding='utf-8') as opened_file:
    text1 = opened_file.read()
```

等价于：

```python
try:
    opened_file = open(file_path)
    text1 = opened_file.read()
finally:
    opened_file.close()
```

无论是否报错，`with` 块结束后文件一定被关闭。

## 内存临时文件

```python
from io import StringIO, BytesIO

temp_file = StringIO()            # 只能存储字符串
temp_file = BytesIO()             # 只能存储字节串
```

```python
temp_file.write('abc')
temp_file.seek(0)                 # 将指针移回起点
temp_file.read()
# 或者
temp_file.getvalue()              # 直接获取全部值，无需移动指针
```

> write 和 read 共用同一指针，写入后需要 `seek(0)` 才能读取。

## pickle 模块

用于将 Python 对象序列化到文件，或从文件反序列化。

```python
import pickle
```

### 序列化（dumps / dump）

```python
data = pickle.dumps({'key1': 12, 'key2': 34})
# => b'\x80\x04\x95...'

# 直接写入文件
with open(r'path', mode='wb') as f:
    pickle.dump(data, f)          # 等价于 f.write(pickle.dumps(data))
```

### 反序列化（loads / load）

```python
data = pickle.loads(encoded_data)

with open(r'path', mode='rb') as f:
    data = pickle.load(f)         # 等价于 pickle.loads(f.read())
```

## 线程

进程负责开辟内存，线程负责在内存中执行命令。一个进程至少有一个主线程。

```python
from threading import Thread

def func(param):
    return

thread1 = Thread(target=func, args=(param,))
thread1.start()
```

### 线程池

```python
from concurrent.futures import ThreadPoolExecutor

with ThreadPoolExecutor(max_workers) as pool:
    pool.submit(func, param)      # 将函数作为线程提交给线程池
```
