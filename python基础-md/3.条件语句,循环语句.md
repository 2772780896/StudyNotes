## 条件判断

```python
if 判断条件1:
    执行语句1
elif 判断条件2:
    执行语句2
else:
    执行语句3
```

- 判断条件为真时执行对应语句块
- `elif` 可以有任意多个
- 所有 `if` / `elif` 都不满足时执行 `else`
- 一旦某个分支被执行，后续分支全部跳过

### 条件的真假

| 假 | 真 |
|----|----|
| `False`、`None`、`0` | `True`，以及除假之外的一切值 |

## 循环遍历

```python
for x in iterable_object:
```

将可迭代对象（list / tuple / dict / string）中的每个元素按顺序赋值给 `x`，直到取完所有元素。

## range() 配合 for 循环

```python
range([start,] end [,step])
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `start` | 起始数字，包含自身 | 0 |
| `end` | 结束数字，不包含自身 | — |
| `step` | 步长 | 1 |

```python
sum = 0
for a in range(1, 101):
    sum = sum + a
```

## 循环控制

```python
continue    # 跳过本次循环剩余语句，进入下一次循环
break       # 立即跳出整个循环
```
