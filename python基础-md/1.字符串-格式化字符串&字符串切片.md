## 格式化字符串

### 百分号格式化字符串

```python
'xxx%d' % (值)       # 使用右侧值替换字符串中的 %d 处
```

| 占位符 | 说明 |
|--------|------|
| `%d` | 替换值为数字类型 |
| `%f` | 替换值为浮点类型 |
| `%s` | 替换值为字符串类型 |

### 字符串.format()

#### 默认指定

```python
"{} {}".format("hello", "world")
# => 'hello world'
```

不指定位置，按默认顺序填入。

#### 索引指定

```python
"{0} {1}".format("hello", "world")
# => 'hello world'

"{1} {0} {1}".format("hello", "world")
# => 'world hello world'
```

#### 命名指定

```python
"{a}, {b}".format(a="hello", b="world")
# => 'hello, world'
```

#### 字典指定

```python
dict = {"a": "hello", "b": "world"}
"{a}, {b}".format(**dict)
# => 'hello, world'
```

#### 列表指定

```python
list = ['hello', 'world']
"{0[0]}, {0[1]}".format(list)    # "0" 必须存在
# => 'hello, world'
```

### f-string

`f'{变量名}'` 会将花括号中的内容作为变量名进行转义替换。

```python
arguments1 = 'hello'
arguments2 = 'world'
f"{arguments1}, {arguments2}"
# => 'hello, world'
```

## 字符串切片

```python
string[num1:num2]    # 获取索引 num1 到 num2-1 的字符
string[:num]         # 获取前 num 个字符
string[:-num]        # 获取去掉后 num 个字符之外的部分
string[num:]         # 获取去掉前 num 个字符之外的部分
string[-num:]        # 获取后 num 个字符
```

## 字符串大小比较

从第一个字符的 ASCII 码值开始逐位比较，直到某位分出大小，则该字符串更大。

```python
'az' > 'abc'    # True
'ac' > 'a'      # True
```
