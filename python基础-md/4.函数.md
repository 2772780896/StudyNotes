## 函数格式

```python
def 函数名(参数1, 参数2, ...):
    函数体
```

## 返回值

函数体中用 `return` 返回值，返回值不能是赋值语句。

```python
变量名 = 函数名()    # 用变量接收返回值
```

如果没有 `return`，函数只是执行过程，没有输出。

## 参数

### 关键字参数

可以不遵循实参与形参的顺序对应。

```python
def print1(a, b):
    print(a, b)

print1(b=1, a=2)    # 输出 2 1
```

### 默认参数

调用时若未传参，则使用默认值。

```python
def print2(a, b=2):
    print(a, b)

print2(0)           # 输出 0 2
```

给默认参数传值时直接按位置传入即可：

```python
print2(a, 9)        # 输出 0 9
```

### 不定长参数

#### *args — 元组形式

变量名前加一个星号 `*`，所有多余的实参以元组形式存放。

```python
def func1(*arguments):
    print(arguments)

func1(5)            # 输出 (5,)
```

- `arguments` 作为元组被调用
- `*arguments` 被调用时保持原始数据格式

#### **kwargs — 字典形式

变量名前加两个星号 `**`，所有多余的关键字参数以字典形式存放。

```python
def func2(**keyword_arguments):
    print(keyword_arguments)

func2(a=4)          # 输出 {'a': 4}
```

- `keyword_arguments` 作为字典 `{关键字: 值}` 被调用
- `**keyword_arguments` 保持原始数据格式

### ** 解包字典参数

```python
def func1(a, b):
    return

d = {'a': 1, 'b': 2}
func1(**d)          # 等价于 func1(a=1, b=2)
```

## 参数传递

### 值传递

当参数是不可变类型（int / str / tuple 等）时，函数内部操作的是参数的副本，不修改原值。

### 引用传递

当参数是可变类型（list / dict 等）时，函数内部直接操作原对象，修改会反映到外部。

### 浅拷贝

若想为可变类型创建独立副本，使用 `list.copy()`：

```python
import copy
l1 = [1, 2, 3]
l2 = l1.copy()
l1.append(4)
print(l2)           # [1, 2, 3]——l2 不受影响
```

## 函数名

函数名本身是一个指向函数对象的变量。

```python
def f():
    pass

print(f)            # <function f at 0x000002D3D3523E20>
```

可以将函数名赋值给另一个变量调用：

```python
def add(a, b):
    print(a + b)

c = add
c(1, 2)             # 输出 3
```

## 作用域

### 作用域层级

```
局部作用域 → 闭包外层 → 全局作用域 → 内建作用域
```

变量查找按此顺序从内向外。只有函数、模块、类会创建新的局部作用域，`if` / `for` / `while` 等不会。

### global

在函数内部声明变量为全局变量。

```python
a = 30

def f():
    a = 10
    def f1():
        global a            # 将 f1() 中所有 a 都变为全局变量
        a = 20
    f1()
    print('1.%d' % a)

f()
print('2.%d' % a)           # 输出：1.20  2.10  3.20？（⚠️ 原文此处示例可能有误）
```

> 注意：同一局部作用域中不允许同时存在分别属于不同作用域的同名变量。
>
> ```python
> a = 0
> def f():
>     a = a + 1   # 报错：局部变量 a 在赋值前被引用
> ```

### nonlocal

若下层函数要修改上层函数的变量（非全局），使用 `nonlocal` 声明：

```python
def outer():
    x = 10
    def inner():
        nonlocal x
        x = 20
    inner()
    print(x)        # 20
```

## 闭包

外层函数返回内层函数，内层函数可以访问外层函数的变量。

```python
def func1():
    def func2():
        pass
    return func2

f = func1()         # f 即 func2，可在 func1 外部调用 func2
```

## 装饰器

### 基本结构

```python
def 装饰器名(func):
    def inner(*args, **kwargs):
        # 调用 func 前后可插入任意操作
        result = func(*args, **kwargs)
        return result
    return inner
```

### @ 语法糖

```python
@装饰器名
def func2():
    return

# 等价于 func2 = 装饰器名(func2)
```

### @装饰器(参数)

```python
@装饰器名(参数)
def func2():
    return

# 等价于 func2 = 装饰器名(参数)(func2)
```

### 多个装饰器的执行顺序

```python
@wrapper1
@wrapper2
@wrapper3
def func4():
    return

# 等价于（从下往上执行）：
# func4 = wrapper3(func4)
# func4 = wrapper2(func4)
# func4 = wrapper1(func4)
```

## 生成器

用 `yield` 替代 `return`，函数即变为生成器。

```python
def 生成器名():
    yield 值

g = 生成器名()
next(g)             # 获取下一个 yield 的输出值
```

生成器的返回值是指向生成器对象的引用，不等于 yield 的输出值。

## 迭代器

- 可被 `next()` 调用，不断产出下一个值
- 可用于 `for` 循环
- 生成器是一种迭代器
