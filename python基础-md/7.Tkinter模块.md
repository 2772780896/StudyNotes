## 创建窗口

```python
from tkinter import Tk

window = Tk()
```

## 窗口操作

```python
window.title('abc')                        # 修改窗口标题
window.geometry('500x500+300+100')         # 修改大小和位置（宽x高+左+上）
window.mainloop()                           # 运行窗口，必须在所有组件装配完成后执行
```

## 标签组件（Label）

用于显示文本和位图。

```python
from tkinter import Label

label = Label(window, text='text')          # 创建标签并设置文本
label.pack()                                # 将组件装配到窗口
```

## 按钮组件（Button）

```python
from tkinter import Button

def function1():
    print(1)

button = Button(window, text='button', command=function1)
button.pack()

button.invoke()                             # 通过代码触发按钮点击
```

## 选择按钮（Checkbutton）

```python
from tkinter import Checkbutton

# 用闭包封装状态，实现点击切换
def function2():
    count = 0
    def print1():
        nonlocal count
        if count % 2 == 0:
            print(2)
        count += 1
    return print1

checkbutton = Checkbutton(window, text='checkbutton', command=function2())
checkbutton.pack()
```

## 文本组件（Text）

用于显示多行文本（可滚动）。

```python

text = Text(window, width=30, height=4)
text.pack()
text.insert('insert', '123')    # 在光标位置插入文本
```

## 列表组件（Listbox）

用于显示多行可选择文本。

```python
from tkinter import Listbox

listbox = Listbox(window, width=30, height=4)
listbox.insert('end', '1', '2', '3')    # 'end' 在末尾添加，'active' 在前端添加
listbox.pack()

listbox.get(1)                 # 获取索引 1 的列表项
listbox.select_includes(1)     # 判断索引 1 是否被选中
listbox.curselection()         # 获取当前选中项的索引
```

## 事件绑定

```python
组件.bind(操作, 事件处理函数)
```

### 操作类型

| 操作 | 含义 |
|------|------|
| `<Button-1>` | 鼠标左键单击 |
| `<Button-2>` | 鼠标滚轮 |
| `<Button-3>` | 鼠标右键单击 |
| `<Double-Button-1>` | 鼠标左键双击 |
| `<B1-Motion>` | 鼠标左键按住移动 |
| `<ButtonRelease-1>` | 鼠标左键松开 |
| `<Key>` | 按键 |
| `<Alt-a>` | 组合按键 |

### 事件函数

```python
def 事件(event):
    pass
```

可用属性：`event.x`、`event.y`（鼠标坐标）、`event.char`（按键名称）。

## 布局

### 绝对布局

```python
label1 = Label(window, text='label1')
label1.place(x=10, y=10)      # 以窗口左上角为原点
```

### 相对布局

**pack 方式**：下一个 `pack()` 的组件在前一个下方，居中对齐。

**Frame 方式**：Frame 透明不可见但占据空间，可嵌套其它组件。

```python
from tkinter import Frame

frame1 = Frame(window, width=2, height=2)
frame1.pack()

frame2 = Frame(frame1, width=1, height=2)
frame2.pack(side='left')      # left / right / top / bottom
```
