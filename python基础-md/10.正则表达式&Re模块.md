## 正则表达式

正则表达式用于用模式匹配普通字符，以 `r''` 声明（`r` 表示 `\` 不转义）。

## 正则字符

| 字符 | 说明 |
|------|------|
| `[chars]` | 匹配 chars 中的任意一个字符 |
| `[^chars]` | 匹配不在 chars 中的任意一个字符 |
| `chars?` | 匹配 0 或 1 个 chars |
| `chars*` | 匹配 0 至无限个 chars |
| `chars*?` | 非贪婪匹配，在起止字符间匹配最短字符串 |
| `chars+` | 匹配 1 至无限个 chars |
| `chars{n}` | 匹配恰好 n 个 chars |
| `chars{n,}` | 匹配 n 至无限个 chars |
| `chars{n,m}` | 匹配 n 至 m 个 chars |
| `chars1\|chars2` | 匹配 chars1 或 chars2 |
| `.` | 匹配除换行符外的任意字符，等价于 `[^\n]` |
| `\d` | 匹配数字字符，等价于 `[0-9]` |
| `\D` | 匹配非数字，等价于 `[^0-9]` |
| `\w` | 匹配字母/数字/下划线，等价于 `[0-9a-zA-Z_]` |
| `\W` | 匹配非字母/数字/下划线 |
| `\s` | 匹配空白符（空格、换行、回车、制表等），等价于 `[ \n\r\f\t]` |
| `\S` | 匹配非空白符 |
| `\特殊字符` | 转义特殊字符，匹配字面值 |

## 匹配模式

| 模式 | 说明 |
|------|------|
| `^chars` | 只在字符串开头匹配 |
| `chars$` | 只在字符串末尾匹配 |

## re 模块

```python
import re
```

### re.search()

```python
re.search(pattern, string, flags=0)
```

只匹配第一个成功的位置。返回 Match 对象或 None。

```python
print(re.search(r'a', 'a$bcdAin sa'))
# => <re.Match object; span=(0, 1), match='a'>
```

`span=(start, end)`，包含 start，不包含 end。

### 分组（group）

```python
re.search(pattern, string).group(组号)
```

- 括号 `()` 中的子模式作为一个组
- 组号按左括号由外向内、由左向右从 1 开始分配
- 组号 0 表示整个匹配

```python
print(re.search(r'a(bc)d', 'abcd').group(1))   # => bc
```

### 命名组

```python
re.search(r'(?P<name>chars)', string).group('name')
```

```python
re.search(pattern, string).groupdict()
# 返回 {组名: 匹配到的字符串}
```

### re.findall()

```python
re.findall(pattern, string, flags=0)
```

匹配所有成功的位置，返回字符串列表。无匹配则返回空列表。

### re.finditer()

```python
re.finditer(pattern, string, flags=0)
```

返回一个生成器，逐个产出 Match 对象。

### re.sub()

```python
re.sub(pattern, repl, string, count=0, flags=0)
```

将匹配到的内容替换为 `repl`，返回替换后的字符串。`count` 指定最大替换次数，0 表示全部替换。
