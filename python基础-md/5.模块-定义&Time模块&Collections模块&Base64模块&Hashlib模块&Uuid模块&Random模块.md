## 模块定义

一个 `.py` 文件即为一个模块。模块可以包含函数、类、变量等。

```python
import 模块名           # 导入整个模块
from 模块名 import 函数  # 导入指定函数
```

### 包

包是包含 `__init__.py` 的目录，用于组织多个模块。

```
a/
  __init__.py
  b.py
  c.py
```

## time 模块

```python
import time
```

| 函数 | 说明 |
|------|------|
| `time.time()` | 返回当前时间戳（1970-01-01 以来的秒数） |
| `time.localtime()` | 返回当前本地时间的 struct_time 对象 |
| `time.localtime(时间戳)` | 将时间戳转为本地 struct_time |
| `time.gmtime()` | 返回当前 UTC 时间的 struct_time |
| `time.gmtime(时间戳)` | 将时间戳转为 UTC struct_time |
| `time.mktime(stru_time)` | 将 struct_time 转为时间戳 |
| `time.sleep(秒)` | 暂停指定秒数 |
| `time.strftime(格式, stru_time)` | 将 struct_time 按格式转为字符串 |
| `time.strptime(字符串, 格式)` | 将字符串按格式解析为 struct_time |
| `time.ctime(时间戳)` | 将时间戳转为可读字符串（无参数则用当前时间） |

### strftime 格式符

| 格式符 | 含义 |
|--------|------|
| `%Y` | 四位数年份 |
| `%m` | 月份（01-12） |
| `%d` | 日（01-31） |
| `%H` | 小时（00-23） |
| `%M` | 分钟（00-59） |
| `%S` | 秒（00-59） |
| `%a` | 简化星期名称 |
| `%A` | 完整星期名称 |

## collections 模块

```python
import collections
```

### namedtuple

创建具有命名字段的元组子类。

```python
Point = collections.namedtuple('Point', ['x', 'y'])
p = Point(1, 2)
p.x    # 1
p.y    # 2
```

### deque

双端队列，支持两端高效插入和删除。

```python
d = collections.deque([1, 2, 3])
d.append(4)
d.appendleft(0)
```

## base64 模块

Base64 将二进制数据编码为 ASCII 字符。

字符集：`A-Z`、`a-z`、`0-9`、`+`、`/`，末尾可能用 `=` 填充。

```python
import base64

base64.b64encode(b'hello')     # 编码
base64.b64decode(编码结果)      # 解码
```

## hashlib 模块

提供常见的哈希算法。

```python
import hashlib

m = hashlib.md5()
m.update(b'hello')
m.hexdigest()       # 十六进制摘要
```

支持的算法：`md5`、`sha1`、`sha256`、`sha512` 等。

## uuid 模块

生成通用唯一标识符（UUID）。

```python
import uuid

uuid.uuid1()        # 基于主机 ID 和当前时间
uuid.uuid4()        # 随机 UUID
```
