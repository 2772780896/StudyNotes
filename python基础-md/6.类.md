## 创建类

```python
class 类名(父类名):
    属性 = 值
    def 方法(self):
        pass
```

- 未指定父类时默认继承 `object`
- `object` 是所有类的基类，提供基础模板

## 创建对象

```python
对象名 = 类名()
```

## 对象方法

### 定义

```python
class MyClass:
    def object_f1(self, x):
        return
```

`self` 代表类的实例对象本身。

### 调用

```python
对象名.方法名(参数)       # 在类框架内/外均可
```

一个对象方法可以调用另一个对象方法：

```python
def object_f2(self, x):
    self.object_f1(x)
```

在类外部也可通过类名调用（需手动传入实例）：

```python
类名.方法名(对象, 参数)
```

## 类方法

### 定义

```python
@classmethod
def class_method(cls, x):
    return
```

`cls` 代表类本身。

### 调用

```python
对象名.方法名(参数)       # 通过对象调用
类名.方法名(参数)          # 通过类名调用
```

## 静态方法

### 定义

```python
@staticmethod
def static_method(x):
    return
```

- 没有 `self` 或 `cls` 参数
- 不能调用实例属性/方法

### 调用

```python
对象名.方法名(参数)
类名.方法名(参数)
```

## 类属性

### 定义

```python
class MyClass:
    class_attr = 1        # 类属性
```

对象创建后，类属性被赋予给对象成为对象属性。

### 调用

```python
类名.属性名               # 类属性可被任意调用
对象名.属性名              # 通过对象调用
```

在类内部的对象方法中，`self.属性名` 是对类属性的引用（相当于 `nonlocal` 效果）。

## 构造方法

```python
class MyClass:
    def __init__(self, 参数):
        self.属性 = 参数    # 创建并初始化对象属性
```

`__init__` 在对象创建时自动调用。

## 继承

Python 支持单继承和多继承。

```python
class Child(Parent):
    pass
```

子类可以重写父类方法。

## 动态类型操作

### type() 动态创建类

```python
MyClass = type('类名', (父类,), {'属性': 值})
```

### 动态修改继承

通过修改 `__bases__` 可改变类的继承关系。

### 内存回收

Python 使用引用计数进行内存管理。当指向某块内存的对象为 0 时，该内存被释放。可通过 `sys.getrefcount()` 查看引用计数。
