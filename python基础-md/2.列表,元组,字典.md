## 列表

### 列表格式

```python
list = [a, b, c, d]
```

### 列表方法

```python
list[x]              # 取索引 x 位置的值，从 0 开始
list[x] = a          # 替换索引 x 位置的值
list.append(a)       # 在末尾添加一个元素
list.insert(a, b)    # 在索引 a 位置插入 b，后续元素顺延
list.extend([4,5,6]) # 将另一个列表的元素追加到末尾
list.pop()           # 从右侧删除一个元素
list.remove(值)      # 删除第一个匹配项。循环可实现全部匹配项删除
list.index(值)       # 返回第一个匹配项的索引
list.count()         # 返回列表中元素的总个数
```

## 元组

### 元组格式

```python
tuple = (1, 2, 3, 4)
```

元组的元素不能被修改。但如果某个元素是列表，该列表内部的元素仍可修改。

### tuple() 转换

```python
tuple(list)    # 将列表转换为元组
```

## 字典

### 字典格式

```python
dic = {key1: value, key2: value, key3: value}    # key 一般为字符串
```

### 字典方法

```python
dic[key]           # 取值，key 不存在会报错
dic.get(key)       # 取值，key 不存在返回空值（前提 key 为字符串）
dic[key4] = value  # 添加新元素
dic.pop(key)       # 删除指定 key 的元素
```
