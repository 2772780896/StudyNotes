# React 基础与 Hooks

React 是一个用于构建用户界面的 JavaScript 库，核心思想是**组件化**和**声明式渲染**。

## 1. 项目启动 (Modern Way)

> **注意**：传统的 `create-react-app` 已不再推荐。

- **推荐方式 (Vite)**：
  ```bash
  npm create vite@latest my-react-app -- --template react
  cd my-react-app
  npm install
  npm run dev
  ```

## 2. JSX 语法

JSX 是 JS 的扩展，允许在 JS 中写类似 HTML 的结构。
- **插值**：`{ expression }`（支持变量、函数调用、三元运算）。
- **条件渲染**：`{ flag && <Component /> }` 或 `{ flag ? <A /> : <B /> }`。
- **列表渲染**：使用 `.map()`，且**必须绑定唯一 `key`**。
  ```jsx
  {list.map(item => <li key={item.id}>{item.name}</li>)}
  ```
- **事件绑定**：
  - `onClick={handleClick}`
  - 传参：`onClick={() => handleClick(id)}`
  - 事件对象：`onClick={(e) => handleClick(e)}`

## 3. 组件通信

- **父传子**：通过 `props` 传递。
- **子传父**：父组件传递一个回调函数给子组件，子组件调用该函数回传数据。
- **跨层级**：使用 `useContext`。
- **插槽**：使用 `props.children`。

## 4. 核心 Hooks (常用)

Hooks 必须在组件顶层调用，不能写在 `if` 或 `for` 循环中。

### 4.1 useState (状态管理)
```javascript
const [count, setCount] = useState(0);
// 更新：setCount(newCount)
```
- **更新机制**：React 18 中是自动批处理的。

### 4.2 useEffect (副作用)
用于处理请求、订阅、DOM 操作等。
```javascript
useEffect(() => {
  // 逻辑代码
  return () => { /* 清理代码，如销毁定时器 */ };
}, [dependency]); // 依赖数组
```
- `[]`：仅在挂载时运行。
- `[prop]`：仅在 prop 变化时运行。

### 4.3 useRef (DOM/持久值)
```javascript
const inputRef = useRef(null);
// 访问：inputRef.current
```
- 修改 `current` 不会触发重新渲染。

### 4.4 useMemo & useCallback (性能优化)
- `useMemo`：缓存**计算结果**。
- `useCallback`：缓存**函数引用**（防止子组件无效重渲染）。

## 5. 样式处理

- **全局导入**：`import './App.css'`。
- **模块化 (CSS Modules)**：`import styles from './App.module.css'`，通过 `className={styles.title}` 引用。

---
**复习提示**：React 的核心是“数据驱动”，状态 (State) 变了，界面就会自动更新。
