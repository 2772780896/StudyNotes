# TypeScript 高级教程

TypeScript 是 JavaScript 的超集，通过引入静态类型检查，显著提升了大型项目的可维护性和开发效率。

## 1. 环境搭建与运行

- **安装**：`npm i -g typescript`
- **手动编译**：`tsc file.ts` -> 生成 `file.js`
- **直接运行 (推荐)**：使用 `ts-node`
  - 安装：`npm i -g ts-node`
  - 运行：`ts-node file.ts`（内存中编译执行，不生成中间文件）

## 2. 核心类型系统

### 2.1 基础类型
- `number`, `string`, `boolean`, `null`, `undefined`, `any` (慎用)
- **联合类型**：`(string | number)`，表示变量可以是其中一种。
- **字面量类型**：`let color: 'red' | 'blue'`，限制变量只能取特定值。

### 2.2 复杂类型
- **数组**：`number[]` 或 `Array<number>`
- **元组 (Tuple)**：`[string, number]`，固定长度和对应位置的类型。
- **函数**：
  ```typescript
  // 声明式
  function add(a: number, b: number): number { return a + b; }
  // 表达式/可选参数
  const greet = (name: string, age?: number): void => { ... }
  ```

## 3. 对象与接口 (Interface)

- **基本定义**：
  ```typescript
  interface User {
    readonly id: number; // 只读
    name: string;
    age?: number; // 可选
  }
  ```
- **继承与交叉**：
  - **继承 (Extends)**：`interface Admin extends User { role: string }`
  - **交叉 (Intersection)**：`type Admin = User & { role: string }`
  - **注意**：如果交叉类型中有同名但冲突的属性，该属性会变为 `never`。

## 4. 类 (Class)

- **成员可见性**：
  - `public` (默认)：到处可访问。
  - `protected`：仅在类及其子类中可访问。
  - `private`：仅在当前类内部可访问。
- **接口实现**：`class MyService implements IService { ... }`

## 5. 泛型 (Generics) - 复用核心

泛型允许在定义时不指定具体类型，而在使用时传入。
- **泛型函数**：`function identity<T>(arg: T): T { return arg; }`
- **泛型约束**：`function getLength<T extends { length: number }>(arg: T) { ... }`

## 6. 内置工具类型 (Utility Types)

| 工具类型 | 描述 | 示例 |
| :--- | :--- | :--- |
| `Partial<T>` | 将 T 中所有属性设为可选 | `Partial<User>` |
| `Required<T>` | 将 T 中所有属性设为必选 | `Required<User>` |
| `Readonly<T>` | 将 T 中所有属性设为只读 | `Readonly<User>` |
| `Pick<T, K>` | 从 T 中挑选一组属性 K | `Pick<User, 'name' \| 'age'>` |
| `Omit<T, K>` | 从 T 中排除一组属性 K | `Omit<User, 'id'>` |
| `Record<K, T>` | 创建一个键为 K、值为 T 的对象类型 | `Record<string, User>` |

## 7. 高级技巧

- **typeof**：获取变量的类型。
- **keyof**：获取对象类型的所有键，返回联合类型。
- **类型断言**：`as Type`，告诉编译器“相信我，它就是这个类型”。
- **类型声明文件 (.d.ts)**：为 JS 提供类型描述。

---
**复习提示**：现代开发中优先使用 `interface` 定义对象，使用 `type` 定义复杂组合。
