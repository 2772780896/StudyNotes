# 22. Vue 3 全攻略

Vue 3 是一款渐进式 JavaScript 框架，带来了更优的性能和更灵活的开发体验。

## 1. 环境搭建与项目启动

- **安装 CLI**：`npm install -g @vue/cli`
- **创建项目**：`vue create 项目名` (项目名需小写并用 `-` 连接)。
- **入口文件**：`src/main.js`。默认导入 `App.vue` 根组件并挂载。

---

## 2. Vue 组件 (.vue 文件)

Vue 组件由三部分组成：`<template>` (结构), `<script>` (逻辑), `<style>` (样式)。

### 2.1 模板语法 (Template)
- **插值**：`{{ 变量名 }}`。
- **属性绑定**：`:src="url"` (简写自 `v-bind:src`)。
- **条件渲染**：
  - `v-if / v-else-if / v-else`：动态创建/销毁 DOM（性能开销较大，适合切换不频繁的场景）。
  - `v-show`：切换 CSS `display` 属性（DOM 始终存在，适合频繁切换）。
- **遍历渲染**：`v-for="item in items" :key="item.id"`。必须指定唯一 key。
- **事件处理**：`@click="method"` (简写自 `v-on:click`)。

### 2.2 脚本逻辑 (Script)

Vue 3 支持两种编写模式：**Options API** (Vue 2 风格) 和 **Composition API** (Vue 3 推荐)。

#### 2.2.1 Options API (选项式)
通过导出一个默认对象来组织代码：
- `name`：组件名称。
- `props`：接收父组件传递的数据。支持类型检查和默认值。
- `data()`：必须是函数，返回一个响应式状态对象。
- `methods`：定义组件的方法。
- `computed`：计算属性，具有缓存性。
- `watch`：监听数据变化并执行副作用。
- **生命周期钩子**：`created`, `mounted`, `updated`, `unmounted` 等。

#### 2.2.2 Composition API (组合式 - 推荐)
使用 `<script setup>` 模式，代码更紧凑，逻辑复用更方便。
- **响应式状态**：
  - `ref()`：用于基本类型或对象。访问需使用 `.value`。
  - `reactive()`：用于对象的深度响应式。可以直接操作。
- **计算属性**：`const fullName = computed(() => ...)`。
- **监听器**：
  - `watch(source, callback)`：显式监听。
  - `watchEffect(callback)`：自动追踪依赖。
- **生命周期**：以 `on` 开头导入，如 `onMounted(() => ...)`。

### 2.3 样式 (Style)
- 支持作用域样式：`<style scoped>`。该样式仅作用于当前组件。

---

## 3. 组件通信

- **父传子**：通过 `props` 传递数据。
- **子传父**：通过 `@事件名` 绑定，子组件使用 `emit` 触发事件。
- **双向绑定**：使用 `v-model`。

---

## 4. Vue 生态 (补充)

- **Vue Router**：官方路由管理器。
- **Pinia**：现代状态管理库（取代了 Vuex），与 Composition API 配合极佳。
- **Vite**：下一代前端构建工具，比传统的 Webpack 速度更快。
