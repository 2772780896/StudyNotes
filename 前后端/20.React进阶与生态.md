# React 进阶与生态

在掌握基础后，进阶开发通常涉及全局状态管理、路由控制和成熟的组件库。

## 1. 全局状态管理：Redux Toolkit (RTK)

RTK 是官方推荐的 Redux 编写方式，大大简化了流程。

### 1.1 定义 Slice (子模块)
```javascript
import { createSlice } from '@reduxjs/toolkit';

const counterSlice = createSlice({
  name: 'counter',
  initialState: { value: 0 },
  reducers: {
    increment: (state, action) => {
      state.value += action.payload || 1; // 支持直接修改 state（内部使用了 Immer）
    }
  }
});

export const { increment } = counterSlice.actions;
export default counterSlice.reducer;
```

### 1.2 异步处理 (Thunk)
```javascript
export const fetchUser = (id) => async (dispatch) => {
  const data = await api.getUser(id);
  dispatch(setUser(data));
};
```

### 1.3 在组件中使用
- `useSelector`：获取状态。
- `useDispatch`：发送 Action。

## 2. 前端路由：React Router v6

### 2.1 路由配置 (推荐方式)
```javascript
const router = createBrowserRouter([
  {
    path: '/',
    element: <Layout />,
    children: [
      { index: true, element: <Home /> },
      { path: 'about/:id', element: <About /> } // 动态路由
    ]
  },
  { path: '*', element: <NotFound /> }
]);
```

### 2.2 核心组件与 Hooks
- `<Outlet />`：渲染子路由的占位符。
- `useNavigate()`：编程式跳转（如 `navigate('/home')`）。
- `useParams()`：获取路径参数。
- `useSearchParams()`：获取查询字符串参数（如 `?id=1`）。

## 3. 路由懒加载 (Optimization)

使用 `lazy` 和 `Suspense` 实现按需加载，减少初始包体积。
```jsx
const Home = lazy(() => import('./pages/Home'));
// 使用时包裹 Suspense
<Suspense fallback={<Loading />}>
  <Home />
</Suspense>
```

## 4. UI 组件库：Ant Design (antd)

企业级 UI 解决方案。
- **常用组件**：`Button`, `Card`, `Form`, `Table`, `Upload`。
- **Form 表单校验**：通过 `onFinish` 获取校验通过后的数据。
- **数据回填**：`form.setFieldsValue(data)`。

## 5. 网络请求：Axios (补充)

虽然 Mockjs 用于模拟，但真实开发首选 Axios。
- **拦截器**：用于统一添加 Token 或处理错误提示。

---
**复习提示**：现代 React 开发通常是 **Vite + React + TS + RTK + React Router + antd** 的组合。
