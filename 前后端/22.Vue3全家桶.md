# Vue 3 全家桶

Vue 3 是一个高性能、易上手的渐进式框架，其核心变革是引入了 **Composition API**。

## 1. 项目搭建

- **推荐方式 (Vite)**：
  ```bash
  npm create vite@latest my-vue-app -- --template vue
  ```
- **传统方式 (Vue CLI)**：`vue create my-project`。

## 2. 核心语法：Composition API

使用 `<script setup>` 是 Vue 3 的标准做法，逻辑更紧凑。

### 2.1 响应式 API
- **`ref()`**：用于定义基本类型或对象。在 JS 中访问需 `.value`，在模板中自动解包。
  ```javascript
  const count = ref(0);
  count.value++;
  ```
- **`reactive()`**：仅用于定义对象或数组的深层响应式。
  ```javascript
  const state = reactive({ name: 'Vue' });
  state.name = 'Vue 3';
  ```

### 2.2 计算属性与监听器
- **`computed()`**：基于依赖缓存计算结果。
- **`watch()`**：精确监听某个数据源的变化。
- **`watchEffect()`**：立即执行并自动追踪依赖。

### 2.3 生命周期钩子
Vue 3 钩子函数需从 `vue` 导入，并以 `on` 开头：
- `onMounted()`, `onUpdated()`, `onUnmounted()`。

## 3. 模板指令

- **数据绑定**：`{{ var }}` (内容), `:prop="val"` (属性)。
- **双向绑定**：`v-model`。
- **条件渲染**：`v-if`, `v-else`, `v-show`。
- **列表渲染**：`v-for="item in list" :key="item.id"`。
- **事件处理**：`@click="method"`。

## 4. 组件通信

- **父传子**：使用 `defineProps`。
- **子传父**：使用 `defineEmits`。
- **依赖注入**：`provide` / `inject` (用于深层嵌套组件)。

## 5. 状态管理：Pinia (补充)

Pinia 是 Vue 3 官方推荐的状态管理库，取代了 Vuex。
- **特点**：极其轻量、支持 TS、没有冗长的 Mutation。
```javascript
export const useCounterStore = defineStore('counter', {
  state: () => ({ count: 0 }),
  actions: {
    increment() { this.count++ }
  }
});
```

## 6. 路由：Vue Router

- **配置**：`createRouter` + `createWebHistory`。
- **跳转**：`<router-link>` 或 `useRouter()` 钩子。
- **参数**：`useRoute()` 获取当前路由信息。

---
**复习提示**：Vue 3 的开发体验非常接近 React Hooks，但其响应式系统更自动化，性能也更优。
