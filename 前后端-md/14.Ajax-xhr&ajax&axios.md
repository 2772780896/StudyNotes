# Ajax（异步通信）

Ajax（Asynchronous JavaScript And XML）在前端与服务器进行异步通信，无需刷新页面即可更新部分数据。

## XMLHttpRequest（XHR）

```javascript
const xhr = new XMLHttpRequest();

xhr.open('请求方法', '目标URL');

xhr.addEventListener('loadend', () => {
    console.log(xhr.response);  // 服务器返回值
});

xhr.send();
```

### GET 请求

参数放在 URL 中：`xhr.open('GET', '/api?key=value')`

### POST 请求

```javascript
// 设置请求头
xhr.setRequestHeader('Content-Type', 'application/json');

// 发送数据
xhr.send('a=1&b=2');                            // 表单格式
xhr.send(JSON.stringify({ a: 1, b: 2 }));        // JSON 格式
```

### 配合 Promise 实现异步

```javascript
const p = new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest();
    xhr.open('GET', '/api/data');
    xhr.addEventListener('loadend', () => {
        if (xhr.status >= 200 && xhr.status < 300) {
            resolve(xhr.response);
        } else {
            reject(xhr.response);
        }
    });
    xhr.send();
});

p.then(result => console.log(result))
 .catch(error => console.log(error));
```

## jQuery Ajax

基于 XHR 和 Promise，局部更新页面。

```javascript
$.ajax({
    url: '/api/submit',
    type: 'POST',       // 或 'GET'
    data: { a: 1, b: 2 },
    success: function(res) {
        // res 为服务器返回值
        console.log(res);
    }
});
```

**批量获取表单数据：**

```javascript
$.ajax({
    url: '/api/submit',
    type: 'POST',
    data: $('#formId').serialize(),
    success: function(res) { /* ... */ }
});
```

## Axios

基于 Ajax 扩展的工具库。

```javascript
axios({
    url: '/api/submit',
    method: 'POST',
    data: { a: 1, b: 2 }
})
.then(res => {
    console.log(res.data);  // 服务器返回值
})
.catch(err => {
    console.log(err);       // 错误信息（可省略）
});
```

### 请求拦截器

```javascript
axios.interceptors.request.use(config => {
    // 请求发送前处理
    return config;
});
```

### 响应拦截器

```javascript
axios.interceptors.response.use(
    response => {
        // 响应处理前
        return response;
    },
    error => {
        return Promise.reject(error);
    }
);
```
