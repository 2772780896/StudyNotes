# JavaScript 基础

## 引入方式

### 内嵌

```html
<script type="text/javascript">
    // JavaScript 代码
</script>
```

HTML5 中 `type` 默认值为 `"text/javascript"`，可省略。`<script>` 一般放在 `<head>` 或 `<body>` 尾部。

### 外部文件

```html
<script src="js文件地址.js"></script>
```

## 注释

```javascript
// 单行注释
/* 多行注释 */
```

## 变量

### 声明

| 关键字 | 作用域 | 重复声明 | 声明前访问 | 可变 |
|------|------|------|------|------|
| `var` | 函数级 | 允许 | `undefined` | 是 |
| `let` | 块级 `{}` | 同一块禁止 | 报错 | 是 |
| `const` | 块级 `{}` | 同一块禁止 | 报错 | 基本类型不可变，数组/对象元素可变 |

```javascript
var a = 1;
let b = 2;
const c = 3;       // 常量
const arr = [1,2]; // arr 的元素仍可修改
```

### 输出与类型查看

```javascript
console.log(变量名);              // 控制台打印
console.log(typeof 变量名);       // 查看数据类型
```

## 字符串

### 方法

| 方法 | 说明 |
|------|------|
| `字符串.length` | 获取长度 |
| `字符串.trim()` | 去除前后空格 |
| `字符串.substring(start, end)` | 截取 [start, end)，不含 end |
| `string1.concat(string2)` | 拼接（无空格） |
| `Number(字符串)` | 字符串转数字 |

### 模板字符串

用反引号 `` ` `` 包裹，`${变量名}` 插入变量：

```javascript
let name = "Alice";
console.log(`Hello, ${name}`);
```

## 对象

### 创建

```javascript
let obj = {
    key1: value1,
    key2: value2
};
// 键名可不加引号
```

### 属性读写

```javascript
obj.key = value;       // 键名自动转字符串
obj["key"] = value;    // 键名需加引号
```

| 方式 | 说明 |
|------|------|
| `obj.key = value` | 修改或新增属性 |
| `delete obj.key` | 删除属性 |
| `delete obj["key"]` | 删除属性（键名需加引号） |

## 数组

数组本质是键名为从 0 开始自增数字的对象。

### 创建

```javascript
let arr = [1, 2, 3];
let arr2 = Array([1, 2, 3]);
let arr3 = Array.from({ length: 5 });   // 5 个 undefined
let arr4 = [...Array(5)];               // 展开稀疏数组
```

> `Array(5)` 创建的是稀疏数组（empty slots），不可遍历，需展开为 `undefined`。

### 方法

| 方法 | 说明 | 返回值 |
|------|------|------|
| `arr.push(v)` | 尾部添加 | 新长度 |
| `arr.unshift(v)` | 头部添加 | 新长度 |
| `arr.pop()` | 尾部删除 | 被删元素 |
| `arr.shift()` | 头部删除 | 被删元素 |
| `arr.splice(i, n)` | 从索引 i 删除 n 个 | 被删元素数组 |
| `arr.splice(i, 0, v)` | 索引 i 处插入 v | `[]` |
| `arr.splice(i, 1, v)` | 索引 i 处替换为 v | 被替换元素 |
| `arr.slice(start, end)` | 截取 [start, end) | 新数组 |

## 遍历

### for...in / for...of

```javascript
// for...in：遍历键名（索引），包括原型链属性
for (let i in arr) { console.log(arr[i]); }

// for...of：直接遍历元素值，忽略原型链
for (let v of arr) { console.log(v); }
```

> `for...of` 中修改 `v` 的引用不影响原数组，需用 `.map()` 生成新数组。

### 数组遍历方法

| 方法 | 说明 | 返回值 |
|------|------|------|
| `arr.map((v, i) => v + 1)` | 每个元素执行回调 | 新数组 |
| `arr.filter(v => v !== 0)` | 按条件过滤 | 新数组 |
| `arr.reduce((prev, cur) => prev + cur)` | 累加（prev=上次返回值, cur=当前值） | 累加结果 |
| `arr.findIndex(v => v === 1)` | 查找满足条件的第一个索引 | 索引或 -1 |
| `arr.find(v => v === 1)` | 查找满足条件的第一个值 | 值或 undefined |
| `arr.forEach((v, i, a) => {})` | 遍历执行操作 | undefined |
| `arr.some(v => 条件)` | 是否有满足条件的元素 | true/false |

> `some` 找到满足项后立即停止遍历。

### 对象遍历

| 方法 | 返回值 |
|------|------|
| `Object.keys(obj)` | 键名数组 |
| `Object.values(obj)` | 值数组 |
| `Object.entries(obj)` | `[[键, 值], ...]` |

### 展开运算符 `...`

```javascript
...arr   // 展开数组每一项，不修改原数组
...obj   // 展开对象每一项，不修改原对象
```

## 排序

```javascript
arr.sort((a, b) => a - b);  // 升序
arr.sort((a, b) => b - a);  // 降序
```

- 默认按字符串 Unicode 码排序
- 返回值为负 → a 排 b 前；正 → a 排 b 后；零 → 不变
- 对象数组：`arr.sort((a, b) => a.属性 - b.属性)`

## 函数

### 定义方式

```javascript
// 1. 函数声明
function fn(params) { 函数体 }

// 2. 函数表达式
const fn = function(params) { 函数体 };

// 3. 箭头函数
const fn = (params) => { 函数体 };
const fn = params => 返回值;        // 单参数、单行 return 简写
const fn = () => { 函数体 };        // 无参数
```

### 参数特性

| 特性 | 说明 |
|------|------|
| `arguments` | 函数内默认变量，存放所有传入参数（包括未命名的） |
| 剩余参数 `...args` | `function fn(...args){}`，`args` 是额外参数组成的数组 |
| 默认参数 | `function fn(a=1){}`，但调用时不能 `fn(a=100)` |
| 对象解构传参 | `const fn = ({a=1, b=2}) => {}`，调用 `fn({a:100, b:200})` |

### 作用域提升（闭包）

```javascript
function outer() {
    function inner() {}
    window.inner = inner;  // 将内部函数暴露到全局
}
```

### 访问外部变量

JavaScript 函数内可直接通过变量名访问并修改外部变量。
