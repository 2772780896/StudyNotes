# Mock.js

## 安装与启用

```bash
npm i mockjs
```

在 React 入口文件 `index.js` 中：

```javascript
import 'mock'
```

## 基本使用

在 `src/mock/index.js` 中：

```javascript
import Mock from 'mockjs'

Mock.mock('/api/path', 'get', {
    code: 200,
    message: 'ok',
    data: { /* 响应数据 */ }
})
```

> 第二个参数（请求方法）可省略，省略时匹配所有方法。

## 响应数据格式

### 随机数量

| 语法 | 说明 | 示例 |
|------|------|------|
| `"list|1-10": []` | 随机 1-10 个空数组 | `list: [[], [], ...]` |
| `"string|1-10": ""` | 随机 1-10 个字符 | |
| `"object|2-4": {a,b,c,d}` | 随机选取 2-4 个属性 | |

### 随机数值

| 语法 | 说明 |
|------|------|
| `"number|1-100": 1` | 随机整数 1-100 |
| `"float|1-100.1-10": 1` | 随机浮点数（整数 1-100，小数 1-10 位） |
| `"boolean|1": true` | 随机布尔值（概率 1/1） |
| `"bool|2-5": false` | false 概率 2/7，true 概率 5/7 |

## Random 随机方法

```javascript
const Random = Mock.Random
```

| 方法 | 说明 | 返回值示例 |
|------|------|------|
| `Random.string(5)` / `@string(5)` | 随机字符串 | `"aB3xY"` |
| `Random.cword(2, 5)` / `@cword(2, 5)` | 随机中文（2-5 字） | `"你好世界"` |
| `Random.email('163.com')` / `@email('163.com')` | 随机邮箱 | `"xxx@163.com"` |
| `Random.boolean()` / `@boolean()` | 随机布尔值 | `true` |
| `Random.integer(60, 100)` / `@integer(60, 100)` | 随机整数 | `73` |
| `Random.float(1, 100, 3, 5)` / `@float(1, 100, 3, 5)` | 随机浮点数 | `58.2341` |
| `Random.datetime('yyyy-MM-dd')` / `@datetime('yyyy-MM-dd')` | 随机日期 | `"2024-03-15"` |
| `Random.time()` / `@time()` | 随机时间 | `"21:33:01"` |
| `Random.color()` | 随机颜色 | `"#3538B2"` |
| `Random.pick(['a','e','i'])` / `@pick(...)` | 随机选取一个 | `"e"` |
| `Random.shuffle(['a','e','i'])` / `@shuffle(...)` | 打乱数组 | `["i","a","e"]` |
| `Random.image('200x100', '#16d46b', '#fff', 'png', 'Hello')` | 占位图 URL | `"http://dummyimage.com/..."` |
| `Random.dataImage('200x100', 'Hello')` | Base64 图片编码 | |

## 获取请求参数

### GET 请求

```javascript
Mock.mock(/^\/api\/path(\?.*)?$/, 'get', function(options) {
    const url = new URL(options.url, 'http://localhost');
    const param = url.searchParams.get('属性');  // 获取查询参数
    return Mock.mock({
        code: 200,
        data: { param }
    });
});
```

### POST 请求

```javascript
Mock.mock(/^\/api\/path(\?.*)?$/, 'post', function(options) {
    const body = JSON.parse(options.body);      // 解析请求体
    return Mock.mock({
        code: 200,
        message: '登录成功',
        _headers: {                              // 设置响应头
            'Content-Type': 'application/json',
            'Set-Cookie': [
                'session_id=abc123; Path=/; HttpOnly',
                'user_pref=dark; Path=/'
            ]
        }
    });
});
```

### 获取请求头

```javascript
const header = options.xhr.requestHeaders;
const value = header['数据名'];
```
