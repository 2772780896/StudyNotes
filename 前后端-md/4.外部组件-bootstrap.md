# Bootstrap

Bootstrap 本质是预定义的 CSS 样式表，通过类选择器提供预设样式。

## 导入

在 `<head>` 中引入：

```html
<link rel="stylesheet" href="../static/plugins/bootstrap-3.4.1-dist/css/bootstrap.css" />
```

## 栅格系统

将页面划分为 12 列，通过给标签指定 class 来控制列宽。

| 类名 | 断点 | 说明 |
|------|------|------|
| `col-xs-n` | 无 | 始终占 n/12 宽度，不随页面变化 |
| `col-sm-n` | < 750px 堆叠 | 占 n/12，窄屏自动换行 |
| `col-md-n` | < 970px 堆叠 | 占 n/12，中等屏幕自动换行 |
| `col-lg-n` | < 1170px 堆叠 | 占 n/12，大屏自动换行 |
| `col-md-offset-n` | — | 向右偏移 n 列（通过 `*` 选择器增加左侧 margin） |

## 容器

| 类名 | 效果 |
|------|------|
| `container` | 居中窄容器，约 `width: 1170px` + 左右 `padding: 15px` + `margin: auto` |
| `container-fluid` | 居中宽容器，约 `width: 1700px` + 左右 `padding: 15px` + `margin: auto` |

## 图标

支持 Font Awesome 等图标库。

## Modal 弹出框

```html
<!-- 触发按钮 -->
<button data-bs-toggle="modal" data-bs-target="#myModal">打开</button>

<!-- 弹框 -->
<div class="modal" id="myModal">
    <button class="btn-close" data-bs-dismiss="modal">关闭</button>
</div>
```

| 属性 | 说明 |
|------|------|
| `data-bs-toggle="modal"` | 点击弹出对话框 |
| `data-bs-dismiss="modal"` | 点击关闭对话框 |

## 分页

设计思路：通过当前页数和每页元素数计算起始位置：

```
起始索引 = 每页元素数 × (当前页数 - 1)
```

通过页数 ±1 切换相邻页码。
