# Django Form、ModelForm 与中间件

## Form

用于在 HTML 中生成表单输入框。

### 定义

```python
from django import forms

class MyForm(forms.Form):
    数据名 = forms.字段类型(
        widget=forms.插件类型(attrs={'class': 'form-control'}),
        required=True,
        initial='初始值'
    )
```

### 标签类型与插件组合

| 标签类型 | 插件类型 | 生成 HTML |
|------|------|------|
| `CharField` | `TextInput` | `<input type="text">` |
| `CharField` | `PasswordInput` | `<input type="password">` |
| `IntegerField` | `NumberInput` | `<input type="number">` |
| `DateField` | `DateInput` | `<input type="date">` |
| `CharField` | `Textarea` | `<textarea>` |

### 模板中使用

```html
<form method="post">
    {% csrf_token %}
    {{ form.数据名 }}
</form>
```

生成标签会自动带上 `id="id_数据名"`。

## ModelForm

根据数据库表自动生成表单，支持数据验证和保存。

### 定义

```python
from django import forms
from app名 import models

class MyModelForm(forms.ModelForm):
    class Meta:
        model = models.表名
        fields = ['列1', '列2']       # 或 '__all__' 使用全部列
        widgets = {
            '列1': forms.TextInput(attrs={'class': 'form-control'})
        }
```

### 视图函数

```python
def view(request):
    if request.method == 'POST':
        form = MyModelForm(data=request.POST)
        if form.is_valid():
            form.save()           # 自动存入数据库
    else:
        form = MyModelForm()

    return render(request, '模板.html', {'form': form})
```

### 获取清洗后数据

```python
data = form.cleaned_data   # 字典，键为列名
```

### 自定义错误

```python
if form.cleaned_data['列名'] != 期望值:
    form.add_error('列名', '错误提示信息')
```

### 模板中遍历

```html
{% for field in form %}
    {{ field }}           <!-- 输入框 -->
    {{ field.errors }}    <!-- 错误信息 -->
{% endfor %}
```

## 中间件

### 创建

在 app 下创建 `middleware/middleware.py`：

```python
from django.utils.deprecation import MiddlewareMixin

class MyMiddleware(MiddlewareMixin):
    def process_request(self, request):
        # 视图执行前运行
        # 无返回值 → 继续执行；有返回值 → 终止后续
        pass

    def process_response(self, request, response):
        # 视图 return 后运行
        return response
```

### 注册

在 `settings.py` 的 `MIDDLEWARE` 中添加：

```python
MIDDLEWARE = [
    # ... 其他中间件
    'app名.middleware.middleware.MyMiddleware',
]
```

> 列表中位置越靠前越先执行，先进后出。

### 豁免特定页面

```python
def process_request(self, request):
    if request.path_info == '/豁免路径/':
        return  # 直接放行，不执行后续逻辑
```
