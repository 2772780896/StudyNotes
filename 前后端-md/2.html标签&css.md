# HTML 标签

## 格式

```html
<开始标签 属性名1="属性值1" 属性名2="属性值2">内容</结束标签>
```

## 标签类型

| 类型 | 说明 |
|------|------|
| 块级标签 | 独占一行，前后内容换行（如 `<div>`、`<h1>`、`<p>`） |
| 内联标签 | 不换行，包裹文档一小部分（如 `<span>`、`<a>`、`<img>`） |
| 空标签 | 只有开始标签，用于插入/嵌入内容（如 `<img />`、`<input />`） |

## 常见标签

### 文档结构

```html
<!DOCTYPE html>          <!-- 声明文档类型为 HTML -->
<html>                   <!-- 根标签 -->
<head>                   <!-- 元数据：标题、字符集、样式、脚本 -->
    <meta charset="utf-8">
    <title>页面标题</title>
</head>
<body>                   <!-- 可见内容 -->
</body>
</html>
```

### 文本标签

| 标签 | 类型 | 说明 |
|------|------|------|
| `<h1> ~ <h6>` | 块级 | 标题，加大加粗 |
| `<p>` | 块级 | 段落 |
| `<div>` | 块级 | 通用容器 |
| `<span>` | 内联 | 行内容器 |

### 链接与图片

```html
<a href="地址" title="提示" target="_blank">链接文字</a>
```

| 属性 | 说明 |
|------|------|
| `href` | 超链接地址 |
| `title` | 鼠标悬停时的提示文字 |
| `target` | `_blank` 新标签页打开，默认当前页 |

```html
<img src="图片地址" alt="替代文字" />
```

| 属性 | 说明 |
|------|------|
| `src` | 图片地址 |
| `alt` | 图片加载失败时显示的文字 |

### 列表

```html
<ul>
    <li>列表项 1</li>
    <li>列表项 2</li>
</ul>
```

### 表格

```html
<table>
    <thead>
        <tr>
            <th>表头1</th>
            <th>表头2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>数据1</td>
            <td>数据2</td>
        </tr>
    </tbody>
</table>
```

| 标签 | 说明 |
|------|------|
| `<table>` | 表格容器 |
| `<thead>` | 表头区域 |
| `<tbody>` | 表体区域 |
| `<tr>` | 行（table row） |
| `<th>` | 表头单元格 |
| `<td>` | 数据单元格 |

### 表单

```html
<input type="text" name="username" placeholder="请输入" value="默认值" readonly />

<textarea rows="5"></textarea>

<select name="city">
    <option value="bj">北京</option>
    <option value="sh">上海</option>
</select>
```

`<input>` 的 `type` 属性：

| type | 说明 |
|------|------|
| `text` | 文本输入框 |
| `password` | 密码框（输入不显示明文） |
| `file` | 文件选择 |
| `radio` | 单选按钮（同 name 互斥） |
| `checkbox` | 复选框（同 name 不互斥） |
| `button` | 普通按钮 |
| `submit` | 提交按钮 |
| `hidden` | 不可见的隐藏字段 |

`<input>` 的通用属性：

| 属性 | 说明 |
|------|------|
| `name` | 数据名称，提交时与 `value` 组成键值对 |
| `value` | 默认值 / 当前值 |
| `placeholder` | 输入框占位提示 |
| `readonly` | 只读 |

`<select>` 中给 `<option>` 设置 `value` 属性实现选项名与值的对应。

### 视频

```html
<video controls poster="封面.jpg">
    <source src="视频.mp4" type="video/mp4" />
</video>
```

| 属性 | 说明 |
|------|------|
| `controls` | 显示播放控件 |
| `poster` | 封面图 |
| `src`（source） | 视频地址 |
| `type`（source） | 视频格式 |

### 注释

```html
<!-- 这是注释 -->
```

## Style 样式属性

在标签上通过 `style` 属性设置，多个属性用 `;` 分隔：

```html
<div style="height: 100px; color: red;">内容</div>
```

### 尺寸

| 属性 | 单位 | 说明 |
|------|------|------|
| `height` | `px` / `%` / `vh` | 高度（`%` 相对父元素，`vh` 相对屏幕） |
| `width` | `px` / `%` / `vw` | 宽度 |

### Display

| 值 | 说明 |
|------|------|
| `block` | 块级，独占一行 |
| `inline` | 内联，宽高内外边距不可用 |
| `inline-block` | 兼具块级和内联特性，可设宽高且同行显示 |
| `none` | 不显示，不占空间，子元素一同隐藏 |

### 颜色与字体

| 属性 | 说明 |
|------|------|
| `color` | 文字颜色（如 `red`、`#ff0000`、`rgb(255,0,0)`） |
| `background-color` | 背景色 |
| `opacity` | 透明度（0-1，0 完全透明） |
| `font-size` | 字体大小（`px`） |
| `font-weight` | 字体粗细 |
| `font-family` | 字体 |

### 对齐与行高

| 属性 | 说明 |
|------|------|
| `text-align: center` | 文字水平居中 |
| `line-height` | 行高，等于容器高度时文字垂直居中 |
| `vertical-align: top` | 同行元素顶部对齐 |

### 浮动

| 属性 | 说明 |
|------|------|
| `float: left` | 左浮动，块级不再独占宽度 |
| `float: right` | 右浮动 |
| `clear: both` | 两侧不允许浮动，出现在所有浮动元素下方（撑开父元素） |

### 内外边距

| 属性 | 说明 |
|------|------|
| `padding-top/left/right/bottom` | 内边距 |
| `margin-top/left/right/bottom` | 外边距 |
| `margin: 上 右 下 左` | 简写四边外边距 |
| `margin: 0 auto` | 左右 auto 时水平居中 |

> `<body>` 标签默认有外边距。

### 定位

| 属性 | 说明 |
|------|------|
| `position: fixed` | 固定在窗口固定位置，通过 `top/left/right/bottom` 定位 |
| `position: relative` | 作为 `absolute` 子元素的参照基准 |
| `position: absolute` | 以最近 `relative` 父元素为基准，通过 `top/left/right/bottom` 定位 |

### 边框

```css
border: 厚度px solid|dashed 颜色;
```

### 鼠标样式

```css
cursor: pointer;
```

### 宽高比

```css
aspect-ratio: 16/9;
```

### Flex 布局

**父容器属性：**

| 属性 | 值 | 说明 |
|------|------|------|
| `display: flex` | — | 启用 Flex 布局 |
| `flex-direction` | `row` / `column` | 主轴方向 |
| `justify-content` | `space-between` / `start` / `end` / `center` | 主轴对齐方式 |
| `flex-wrap` | `wrap` / `nowrap` | 是否换行 |

**子元素属性：**

| 属性 | 说明 |
|------|------|
| `flex-grow: 0/1` | 是否可放大 |
| `flex-shrink: 1/0` | 是否可缩小（默认 1） |
| `flex-basis: 100px` | 基础宽度（默认 auto） |
| `flex: 0 1 auto` | 三属性简写 |
| `flex: 3` | 等价 `flex: 3 1 auto`，与 `flex: 1` 搭配可比例分配空间 |

### Grid 布局

**父容器属性：**

| 属性 | 说明 |
|------|------|
| `display: grid` | 启用 Grid 布局 |
| `grid-template-columns: 200px 100px 200px` | 三列宽度 |
| `grid-template-rows: 50px 60px` | 两行高度 |
| `grid-template-rows: repeat(2, 50px)` | 指定行数的高度 |
| `grid-template-columns: repeat(auto-fill, 200px)` | 尽可能按指定宽度容纳子元素 |
| `grid-template-columns: 1fr 2fr` | 按比例分配宽度 |
| `grid-template-columns: 1fr minmax(100px, 2fr)` | 设置最小/最大宽度 |
| `grid-template-columns: 100px auto 100px` | `auto` 自动填充剩余空间 |
| `row-gap: 10px` | 子元素上下间距 |
| `column-gap: 20px` | 子元素左右间距 |
| `gap: 10px 20px` | 间距简写 |
| `justify-items: start/end/center` | 同列子元素水平对齐 |
| `align-items: start/end/center` | 不同行垂直对齐 |

**子元素属性：**

| 属性 | 说明 |
|------|------|
| `grid-column: 1/3` | 从第 1 列到第 3 列 |
| `grid-column: span 2` | 占据 2 列 |
| `justify-self: start/end/center` | 当前元素在单元格内水平对齐 |

# CSS

层叠样式表（Cascading Style Sheets），统一管理样式。

## 写在 HTML 内

在 `<head>` 的 `<style>` 标签中编写（注释用 `/* */`）：

```html
<head>
    <style>
        /* CSS 注释 */
        选择器 {
            属性1: 值1;
            属性2: 值2;
        }
    </style>
</head>
```

### 选择器

| 选择器 | 语法 | 说明 | 示例 |
|------|------|------|------|
| 类选择器 | `.类名 { }` | 匹配 `class="类名"` 的元素 | `.box { color: red; }` |
| 标签选择器 | `标签名 { }` | 匹配所有该标签 | `p { font-size: 14px; }` |
| 属性选择器 | `标签名[属性=值] { }` | 匹配指定属性的标签 | `input[type="text"] { }` |
| 后代选择器 | `.类名 标签名 { }` | 匹配类内所有该标签 | `.box p { }` |

使用方式：

```html
<div class="box">应用 .box 样式</div>
```

## 写在外部 CSS 文件

将样式写入 `.css` 文件，在 HTML 的 `<head>` 中通过 `<link>` 引入：

```html
<head>
    <link rel="stylesheet" href="样式.css" />
</head>
```

| 属性 | 说明 |
|------|------|
| `rel="stylesheet"` | 指定引入文件类型为样式表 |
| `href` | CSS 文件路径 |
