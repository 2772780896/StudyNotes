# DOM、jQuery 与 BOM

## DOM

### 获取标签

| 方法 | 说明 |
|------|------|
| `document.getElementById("id")` | 按 id 获取（重复 id 只取第一个） |
| `document.querySelector("选择器")` | 按 CSS 选择器获取第一个 |
| `document.querySelectorAll("选择器")` | 按 CSS 选择器获取全部（返回 NodeList） |
| `document.getElementsByTagName("标签名")` | 按标签名获取（返回 NodeList） |

**获取标签后是引用传递**，修改该变量会直接影响页面标签。

**层级关系：**

| 方法 | 说明 |
|------|------|
| `标签.parentNode` | 获取父标签 |
| `标签.children` | 获取子标签（NodeList） |
| `标签.children[索引]` | 按索引获取子标签 |
| `标签.children.querySelector("选择器")` | 在子标签中筛选（id 选择器会全局查找，不限于子标签） |

```javascript
// 在子标签中按标签名查找（变通方法）
for (let el of parent.children) {
    if (el.tagName === "DIV") { /* ... */ }
}
```

### 获取标签内容

```javascript
let text = 标签.innerText;   // 读取
标签.innerText = "新内容";     // 修改
```

### 创建标签

```javascript
let el = document.createElement("span");
父标签.appendChild(el);
```

### 创建图片标签（base64）

```javascript
import imgObj from '图片路径';
let img = document.createElement("img");
img.src = imgObj;
父标签.appendChild(img);
```

### 事件绑定

```html
<!-- 方式一：内联 -->
<input type="button" onclick="fn()" />
```

```javascript
// 方式二：事件监听器
标签.addEventListener('click', function(event) {
    // event.target 为触发事件的标签
});
```

| 事件类型 | 说明 |
|------|------|
| `click` | 点击 |
| `change` | 值变化 |
| `mouseover` | 鼠标移入 |

### 表单批量获取

```javascript
const data = serialize(form, { hash: true, empty: true });
```

| 参数 | 说明 |
|------|------|
| `hash: true` | 返回字典对象 |
| `hash: false` | 返回 `key1=val1&key2=val2` 字符串 |
| `empty: true` | 空值也获取 |

### Cookie

```javascript
// 设置（expires 省略时浏览器关闭即删除；path 省略时当前域名全路径生效）
document.cookie = "name=value; expires=Thu, 18 Dec 2043 12:00:00 GMT; path=/";

// 读取
let cookies = document.cookie;
```

## jQuery

### 导入

```html
<script src="../static/js/jquery-3.7.1.js"></script>
```

### 选择器

| 语法 | 说明 |
|------|------|
| `$("#id")` | 按 id |
| `$(".class")` | 按类 |
| `$("tag")` | 按标签名 |
| `$("tag[attr='val']")` | 按属性 |
| `$(".class tag")` | 后代选择器 |
| `$("tag.class")` | 标签 + 类复合 |
| `$("#a, #b")` | 同时多个 |

### 层级遍历

| 方法 | 说明 |
|------|------|
| `$("sel").prev()` | 上一个兄弟标签 |
| `$("sel").next()` | 下一个兄弟标签 |
| `$("sel").siblings()` | 所有兄弟标签 |
| `$("sel").siblings(".class")` | 兄弟标签中筛选 |
| `$("sel").parent()` | 父标签 |
| `$("sel").children()` | 子标签 |
| `$("sel").children(".class")` | 子标签中筛选 |
| `$("sel").find(".class")` | 所有后代中查找 |

### 操作标签

| 方法 | 说明 |
|------|------|
| `$("tag").text("内容")` | 创建标签并设置文本 |
| `$("父").append(el)` | 添加子标签 |
| `$("sel").remove()` | 删除标签 |

### 类操作

| 方法 | 说明 |
|------|------|
| `$("sel").hasClass("c")` | 判断是否有类，返回 true/false |
| `$("sel").addClass("c")` | 添加类 |
| `$("sel").removeClass("c")` | 移除类 |

### Input 操作

```javascript
$("sel").val();       // 获取值
$("sel").val("新值");  // 修改值
```

### 事件绑定

```javascript
$("sel").click(function() {
    // $(this) 表示当前被点击的标签
});
```

### 遍历

```javascript
$.each(obj, function(key, value) {
    // 遍历对象每一项
});
```

## BOM

### window 对象

| 属性/方法 | 说明 |
|------|------|
| `window.innerHeight` | 窗口内部高度 |
| `window.innerWidth` | 窗口内部宽度 |
| `window.open()` | 打开新窗口 |
| `window.close()` | 关闭当前窗口 |
| `window.moveTo()` | 移动窗口 |
| `window.resizeTo()` | 调整窗口尺寸 |

### location（可省略 `window.`）

| 属性 | 说明 |
|------|------|
| `location.href` | 当前页面 URL |
| `location.hostname` | 域名 |
| `location.pathname` | 路径和文件名 |
| `location.port` | 端口 |
| `location.protocol` | 协议（http/https） |

### 弹框（可省略 `window.`）

| 方法 | 说明 | 返回值 |
|------|------|------|
| `alert("内容")` | 提示弹框 | — |
| `confirm("内容")` | 确认弹框（确定/取消） | `true` / `false` |
| `prompt("内容", "默认值")` | 输入弹框 | 输入值 或 `null` |

### 定时器（可省略 `window.`）

| 方法 | 说明 |
|------|------|
| `setInterval(fn, ms)` | 每 ms 毫秒执行一次 fn |
| `clearInterval(id)` | 清除定时器 |
| `setTimeout(fn, ms)` | ms 毫秒后执行一次 fn |
| `clearTimeout(id)` | 清除定时器 |

### localStorage（可省略 `window.`）

```javascript
localStorage.setItem('key', 'value');  // 存入（无过期时间，不可跨域）
localStorage.getItem('key');           // 读取
```
