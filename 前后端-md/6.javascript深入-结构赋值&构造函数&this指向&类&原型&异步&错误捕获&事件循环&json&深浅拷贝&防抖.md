# JavaScript 深入

## 数组解构赋值

基于数组顺序依次赋值：

```javascript
let arr = [1, 2, 3];

const [a, b, c] = arr;       // a=1, b=2, c=3
const [a, , c] = arr;        // a=1, c=3  跳过中间
const [a, ...rest] = arr;    // a=1, rest=[2, 3]
```

## 对象解构赋值

```javascript
const obj = { name: 'Alice', age: 25 };

const { name, age } = obj;                         // 变量名需与键名一致
const { name: a, age: b } = obj;                   // 重命名 a='Alice', b=25
const { name = '', age = 0, height = 180 } = obj;  // 设默认值
```

> `{ name }` 是 `{ name: name }` 的简写。属性顺序不要求一致。

## 构造函数

```javascript
function Person(name) {
    this.name = name;
    // 无需 return，return 无效
}

const p = new Person('Alice');  // { name: 'Alice' }
```

- `new` 返回对象，`this` 指向新对象
- 可通过 `实例.属性 = 值` 添加新属性

### 静态属性 / 方法

```javascript
Person.staticProp = 'static';  // 构造函数内或外部定义
Person.staticProp;              // 通过构造函数名访问

// 静态方法中的 this 指向构造函数本身
```

## this 指向

| 场景 | this 指向 |
|------|------|
| `function` 声明函数 | 调用者；全局调用时指向 `window`（严格模式 `undefined`） |
| 箭头函数 | 不产生自己的 this，继承来自上一级的 this |

### 改变 this 指向

| 方法 | 语法 | 说明 |
|------|------|------|
| `call` | `fn.call(obj, arg1, arg2)` | this 指向 obj 并立即调用 |
| `apply` | `fn.apply(obj, [arg1, arg2])` | 同 call，但参数用数组包裹 |
| `bind` | `const newFn = fn.bind(obj)` | 返回 this 绑定到 obj 的新函数，不立即执行 |

## 类 class

构造函数的一种语法糖。

```javascript
class Person {
    constructor(name) {
        this.name = name;    // 初始化
    }
    say() {                  // 方法
        console.log(this.name);
    }
}

const p = new Person('Alice');
p.say();                     // 'Alice'
```

## 原型

### 基本概念

| 属性 | 所在位置 | 说明 |
|------|------|------|
| `prototype` | 构造函数 | 构造函数的原型对象 |
| `__proto__` | 实例对象 | 指向其构造函数的 `prototype` |
| `constructor` | 原型对象 | 指向所属构造函数 |

```javascript
Person.prototype.greet = function() {};  // 所有实例共享该方法
```

> 将方法定义在原型上可避免每个实例重复创建方法，节省内存。

### 原型链

- `构造函数.prototype` 也是对象，其 `__proto__` 指向 `Object.prototype`
- `Object.prototype.__proto__` 为 `null`（终点）
- 访问属性时沿原型链逐级向上查找

```javascript
arr instanceof Array   // 判断构造函数 prototype 是否在原型链上
```

### 原型继承

```javascript
Child.prototype = new Parent();
Child.prototype.constructor = Child;  // 将 constructor 指回自身
```

### 扩展内置类型

```javascript
Array.prototype.sum = function() {
    return this.reduce((a, b) => a + b, 0);
};
```

## 异步

### Promise

```javascript
const p = new Promise((resolve, reject) => {
    if (成功) {
        resolve(结果);
    } else {
        reject(错误);
    }
});

p.then(result => {
    console.log(result);   // resolve 传入的值
}).catch(error => {
    console.log(error);    // reject 传入的值
});
```

- `.then()` 和 `.catch()` 均返回 Promise，可链式调用

### async / await

基于 Promise 的语法糖。

```javascript
async function fn() {
    const result = await 异步函数();
    return result;
}

// 箭头函数
const fn = async () => {
    const result = await 异步函数();
};
```

- `async` 函数始终返回 Promise
- `await` 暂停函数执行，等待 Promise 完成；同时外部代码继续执行

## JSON


## 浅拷贝与深拷贝

### 浅拷贝

创建新对象但内部嵌套对象仍共享同一内存。

```javascript
// 方式一：展开运算符
const copy = { ...obj };

// 方式二：Object.assign
const copy = {};
Object.assign(copy, obj);
```

### 深拷贝

嵌套对象也复制为新对象。

```javascript
// 方式一：JSON（无法处理函数、undefined、循环引用）
const copy = JSON.parse(JSON.stringify(obj));

// 方式二：lodash
const _ = require('lodash');
const copy = _.cloneDeep(obj);

// 方式三：递归
function deepClone(obj) {
    if (typeof obj !== 'object' || obj === null) return obj;
    const copy = Array.isArray(obj) ? [] : {};
    for (let key in obj) {
        copy[key] = deepClone(obj[key]);
    }
    return copy;
}
```

## 防抖 debounce

函数连续触发时，只执行最后一次。每次触发重置计时器：

```javascript
function debounce(fn, delay = 300) {
    let timer;
    return function (...args) {
        clearTimeout(timer);
        timer = setTimeout(() => fn.apply(this, args), delay);
    };
}

// lodash
_.debounce(fn, 300);
```

## 节流 throttle

函数连续触发时，固定时间间隔内只执行一次：

```javascript
function throttle(fn, delay = 300) {
    let last = 0;
    return function (...args) {
        const now = Date.now();
        if (now - last >= delay) {
            last = now;
            fn.apply(this, args);
        }
    };
}
```
