# Django 基础

## 创建项目

```bash
cd 目标目录
C:\Users\...\django-admin.exe startproject 项目名
```

## 创建 App

```bash
cd 项目目录                # manage.py 所在目录
python manage.py startapp app名
```

### App 常用文件

| 文件 | 用途 |
|------|------|
| `urls.py` | URL 与视图函数对应关系 |
| `views.py` | 视图函数 |
| `models.py` | 数据库模型 |

### 注册 App

在 `settings.py` 的 `INSTALLED_APPS` 列表中添加 `'app名'`。

## URL 配置

```python
# urls.py
from app名 import views

urlpatterns = [
    path('路径/', views.函数名),
]
```

### 按模块分类管理

在 app 目录下创建 `views/` 文件夹，每个模块一个 `.py` 文件：

```python
# urls.py
from app名.views import 分类名

urlpatterns = [
    path('路径/', 分类名.函数名),
]
```

### 动态 URL 传参

```python
path('page/<int:id>/detail/', views.函数名)

# views.py
def 函数名(request, id):
    # id 为 URL 中传入的数字
    return ...
```

## 视图函数

```python
from django.shortcuts import HttpResponse, render, redirect

def 函数名(request):
    # ...
```

### 返回值

| 返回方式 | 说明 |
|------|------|
| `HttpResponse('字符串')` | 返回纯文本 |
| `render(request, '模板.html')` | 返回 HTML（模板在 `app/templates/` 下） |
| `redirect('https://...')` | 重定向到其他网址 |

## 数据接收

```python
def 函数名(request):
    if request.method == 'GET':
        data = request.GET
    elif request.method == 'POST':
        data = request.POST
```

- `request.GET` / `request.POST` 类型为 `QueryDict`（类似不可变多键字典）
- 同名多值时值为列表：`name: [value1, value2]`

## 数据提交

```html
<form method="POST" action="/路径/">
    {% csrf_token %}
    <input name="字段名" value="值" />
    <input type="submit" />
</form>
```

| 属性 | 说明 |
|------|------|
| `method` | `GET`（URL 可见）或 `POST`（URL 不可见） |
| `action` | 提交目标地址（Django 中需以 `/` 结尾，省略则提交到当前地址） |

### CSRF 处理

Django 默认开启 CSRF 保护，需在 `<form>` 中添加 `{% csrf_token %}`。

豁免单个视图：

```python
from django.views.decorators.csrf import csrf_exempt

@csrf_exempt
def 函数名(request):
    return ...
```

## 启动

```bash
python manage.py runserver
```

## 静态文件

静态文件（图片、CSS、JS 等）放在 `app/static/` 下，按类型分目录（`img/`、`css/`、`js/`、`plugins/`）。

### 模板中引用

```html
{% load static %}

<link rel="stylesheet" href="{% static 'css/style.css' %}" />
<img src="{% static 'img/logo.png' %}" />
```

## 模板语法

### 变量

```python
# views.py
return render(request, '模板.html', {'变量名': 值})
```

```html
<!-- 模板.html -->
{{ 变量名 }}
```

### 列表与字典

```html
<!-- 列表索引 -->
{{ 列表名.索引 }}

<!-- 字典取值 -->
{{ 字典名.键名 }}

<!-- 循环列表 -->
{% for item in 列表名 %}
    {{ item }}
{% endfor %}

<!-- 循环字典 -->
{% for key, value in 字典名.items %}
    {{ key }}: {{ value }}
{% endfor %}
```

### 条件

```html
{% if 条件 %}
    ...
{% elif 条件 %}
    ...
{% else %}
    ...
{% endif %}
```

### 函数调用

在 `{{ }}` 中调用函数时不写括号，模板自动添加。

### 管道过滤器

```html
{{ 'hello' | upper }}
{{ value | default:'默认值' }}
```

### 渲染 HTML 字符串

默认字符串会被转义，需用 `mark_safe`：

```python
from django.utils.safestring import mark_safe

return render(request, '模板.html', {'变量': mark_safe('<div>内容</div>')})
```

## 模板继承

**父模板：**

```html
<!-- base.html -->
<html>
<body>
    {% block content %}{% endblock %}
</body>
</html>
```

**子模板：**

```html
{% extends 'base.html' %}

{% block content %}
    子模板独有内容
{% endblock %}
```

## Cookie 与 Session

### Django Session

```python
def 视图(request):
    # 设置（自动写入 django_session 表）
    request.session['key'] = 'value'

    # 获取
    value = request.session.get('key')

    # 清除
    request.session.clear()
```

### 原生 Cookie

```python
def 视图(request):
    # 设置
    response = render(request, '模板.html')
    response.set_cookie('key', 'value')
    return response

    # 获取
    value = request.COOKIES.get('key')
```
