# Django ORM

ORM（Object-Relational Mapping）将 Python 类映射为数据库中的表。

## 数据库配置

在 `settings.py` 中：

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': '数据库名',
        'USER': '用户名',
        'PASSWORD': '密码',
        'HOST': '127.0.0.1',
        'PORT': '3306',
    }
}
```

## 定义模型

在 app 的 `models.py` 中：

```python
from django.db import models

class 表名(models.Model):
    列名 = models.字段类型(参数)
```

### 常用字段类型

| 类型 | 参数 | 说明 |
|------|------|------|
| `CharField` | `max_length=N` | 字符串 |
| `IntegerField` | — | 整数 |
| `DecimalField` | `max_digits=M, decimal_places=D` | 小数 |
| `SmallIntegerField` | — | 小整数 |

### 字段参数

| 参数 | 说明 |
|------|------|
| `primary_key=True` | 主键 |
| `default=值` | 默认值 |
| `null=True, blank=True` | 允许空值 |
| `choices=选项元组` | 选择约束 |

### 外键关联

```python
class Author(models.Model):
    name = models.CharField(max_length=64)

class Book(models.Model):
    author = models.ForeignKey(
        to='Author',
        to_field='id',
        on_delete=models.CASCADE
    )
```

实际数据库列名为 `author_id`。

| `on_delete` | 说明 |
|------|------|
| `CASCADE` | 级联删除 |
| `PROTECT` | 有引用则禁止删除 |
| `SET_NULL` | 置空（需 `null=True`） |
| `SET_DEFAULT` | 置默认值（需 `default=值`） |

### 选择约束

```python
STATUS_CHOICES = ((1, '启用'), (2, '禁用'))

class Item(models.Model):
    status = models.SmallIntegerField(choices=STATUS_CHOICES)

# 获取选项显示值
item.get_status_display()  # '启用' 或 '禁用'
```

## 迁移

```bash
python manage.py makemigrations
python manage.py migrate
```

新增列时需提供默认值或允许空值：

```python
列名 = models.CharField(max_length=32, default='默认值')
列名 = models.CharField(max_length=32, null=True, blank=True)
```

## CRUD 操作

```python
from app名 import models
```

### 创建

```python
models.表名.objects.create(列1=值1, 列2=值2)
```

### 查询

| 方法 | 说明 |
|------|------|
| `models.表名.objects.all()` | 全部数据（QuerySet） |
| `models.表名.objects.filter(列=值)` | 条件筛选 |
| `models.表名.objects.exclude(列=值)` | 排除条件 |

```python
# QuerySet 转字典
queryset.values('列1', '列2')

# QuerySet 转列表
queryset.values_list('列1', '列2')

# 排序
queryset.order_by('列名')    # 升序
queryset.order_by('-列名')   # 降序
```

**不等式条件：**

| 后缀 | 说明 |
|------|------|
| `__gt` | 大于 |
| `__gte` | 大于等于 |
| `__lt` | 小于 |
| `__lte` | 小于等于 |

**字符串条件：**

| 后缀 | 说明 |
|------|------|
| `__startswith` | 以...开头 |
| `__endswith` | 以...结尾 |
| `__contains` | 包含 |

```python
models.表名.objects.filter(列__gt=10)
models.表名.objects.filter(列__contains='关键词')
models.表名.objects.filter(列=值).exists()  # 是否存在
```

### 修改

```python
models.表名.objects.filter(条件列=值1).update(修改列=值2)
```

### 删除

```python
models.表名.objects.filter(列=值).delete()
models.表名.objects.all().delete()  # 删除全部
```

## SQLite（Django 默认）

```python
import sqlite3

conn = sqlite3.connect('数据库.db')   # 不存在则创建
cursor = conn.cursor()

cursor.execute('SQL语句')
result = cursor.fetchall()          # [(列1, 列2), ...]

cursor.close()
conn.close()
```
