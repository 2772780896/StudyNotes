## 基本结构

```python
from flask import Flask, render_template, request

app = Flask(__name__)


@app.route('/show/info')
def index():
    return render_template('测试1（数据的提交）.html')
```

`@app.route('/show/info')` 等价于 `index = app.route('/show/info')(index)`，将 URL 路径与视图函数绑定。当访问 `/show/info` 时，Flask 向 `index` 函数发送 GET 请求，函数的返回值作为响应。

## render_template

用于渲染 HTML 模板。默认从与当前 `.py` 同级目录下的 `templates/` 文件夹中查找。

```python
return render_template('html文件')
```

修改默认模板路径：

```python
app = Flask(__name__, template_folder='路径')
```

## 启动

```python
if __name__ == '__main__':
    app.run()
```

使网站持续运行。

## Jinja2 模板语法

### 变量

在 HTML 中用 `{{ 变量名 }}` 作为占位符：

```python
return render_template('html文件', 变量名=值)
```

### 循环

```html
{% for i in 列表名 %}
    {{ i }}
{% endfor %}
```

```python
return render_template('html文件', 列表名=列表)
```

### 条件

```html
{% if 条件 %}
    ...
{% elif 条件 %}
    ...
{% else %}
    ...
{% endif %}
```
