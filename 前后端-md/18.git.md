# Git

## 安装与配置

```bash
git -v                                      # 查看版本
git config --list                           # 查看配置
git config --global user.name "用户名"        # 设置用户名
git config --global user.email "邮箱"         # 设置邮箱
```

## 创建仓库

```bash
git init       # 在当前目录创建 .git 仓库，当前目录即为工作区
```

## 暂存区操作

```bash
git add 文件路径         # 添加文件到暂存区（. 表示所有改动文件）
git add .               # 添加所有文件
git ls-files            # 查看暂存区文件
git rm --cached 文件路径  # 从暂存区移除
```

## 提交到版本库

```bash
git commit -m "注释"    # 提交暂存区到版本库
git log --oneline       # 查看提交记录
```

> 若子目录中存在 `.git` 仓库，即使删除 `.git` 也会被忽略，需先清除子目录缓存。

## 文件状态

```bash
git status -s    # 查看文件状态
```

| 标识 | 含义 |
|------|------|
| `U` | 未被 Git 跟踪过 |
| `A` | 第一次被暂存 |
| `''`（空） | 工作区、暂存区、版本库三者一致 |
| `M` | 已修改 |

## 数据回溯

| 命令 | 说明 |
|------|------|
| `git restore 文件` | 工作区回溯为暂存区内容 |
| `git reset --soft 版本号` | 回溯到指定版本，保留工作区和暂存区（不影响 U 文件） |
| `git reset --hard 版本号` | 清空工作区和暂存区，回溯到指定版本 |
| `git reset --mixed 版本号` | 清空暂存区，回溯到指定版本 |

## 文件忽略

创建 `.gitignore` 文件，写入要忽略的文件名/通配符，提交时自动跳过。

## 分支

默认主分支为 `master`（或 `main`）。

```bash
git branch 分支名       # 创建分支（从当前版本库最新版延伸）
git checkout 分支名     # 切换分支（HEAD 指针指向目标分支）
git branch              # 查看所有分支
git branch -d 分支名    # 删除分支
```

### 合并

```bash
git merge 分支名        # 将目标分支合并到当前分支
```

> 合并时若两个分支修改了同一数据，会产生**冲突**，需手动选择保留方案。

## 远程仓库

### 绑定

```bash
git remote add 仓库别名 仓库地址     # 绑定
git remote remove 仓库别名           # 取消绑定
git remote -v                        # 查看已绑定仓库
```

### 推送

```bash
git push -u 仓库别名 本地分支:远程分支
```

- 首次推送需验证账号密码（凭据在 控制面板 → 凭据管理器 → Windows 凭据 中管理）
- 远程有新内容时需先拉取才能推送

### 拉取

```bash
git clone 仓库地址                           # 克隆（自动绑定远程仓库）
git pull 仓库别名 远程分支:本地分支            # 拉取并合并
git pull --rebase 仓库别名 远程分支            # 强制拉取（当本地与远程无共同祖先时）
```

## 代理设置

```bash
git config --global http.proxy http://地址      # HTTP 代理
git config --global https.proxy https://地址    # HTTPS 代理
git config --global https.proxy socks5://地址   # SOCKS5 代理
git config --global --get https.proxy           # 查看代理
git config --global --unset https.proxy         # 取消代理
```
