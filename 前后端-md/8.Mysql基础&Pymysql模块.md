# MySQL 基础 & PyMySQL

## 安装与环境

- 下载：https://downloads.mysql.com/archives/installer/
- 安装：Custom → MySQL Servers → Advanced Options → 配置路径和密码
- 环境变量：添加 `bin` 目录到 Path

## 连接与基本操作

```bash
# 连接
mysql -u root -p

# 设置密码
mysql> set password = password('密码');

# 退出
mysql> exit
```

## 数据库操作

```sql
-- 查看所有数据库
SHOW DATABASES;

-- 创建数据库
CREATE DATABASE 数据库名 DEFAULT CHARSET utf8 COLLATE utf8_general_ci;

-- 删除数据库
DROP DATABASE 数据库名;

-- 进入数据库
USE 数据库名;
```

> 数据库相当于文件夹，数据表相当于文件。

## 数据表操作

```sql
-- 查看所有表
SHOW TABLES;

-- 创建表
CREATE TABLE 表名 (
    列名1 数据类型 特殊参数,
    列名2 数据类型 特殊参数
) DEFAULT CHARSET=utf8;

-- 查看表结构
DESC 表名;

-- 重命名表
ALTER TABLE 旧表名 RENAME 新表名;

-- 删除表
DROP TABLE 表名;

-- 修改列
ALTER TABLE 表名 CHANGE 原列名 新列名 数据类型 特殊参数;

-- 添加列
ALTER TABLE 表名 ADD 列名 数据类型 特殊参数 AFTER 已有列名;

-- 删除列
ALTER TABLE 表名 DROP 列名;
```

### 数据类型

| 类型 | 说明 | 范围 |
|------|------|------|
| `INT` | 有符号整数 | -2147483648 ~ 2147483647 |
| `INT UNSIGNED` | 无符号整数 | 0 ~ 4294967295 |
| `TINYINT` | 有符号小整数 | -128 ~ 127 |
| `TINYINT UNSIGNED` | 无符号小整数 | 0 ~ 255 |
| `DECIMAL(M, D)` | 精确小数，M 为总位数（≤65），D 为小数位数（≤30） | — |
| `CHAR(N)` | 定长字符串，固定占用 N 个字符空间 | N ≤ 255 |
| `VARCHAR(N)` | 变长字符串，按实际长度占用，不超过 N | ≤ 65535 字节 |
| `TEXT` | 长文本字符串 | ≤ 65535 字符 |
| `DATE` | 日期 | YYYY-MM-DD |
| `DATETIME` | 日期时间 | YYYY-MM-DD HH:MM:SS |

### 特殊参数

| 参数 | 说明 |
|------|------|
| `NOT NULL` | 不允许为空 |
| `DEFAULT 值` | 默认值 |
| `PRIMARY KEY` | 主键（不允许为空、不允许重复） |
| `AUTO_INCREMENT` | 自增（每行自动 +1） |

## 数据操作

```sql
-- 插入
INSERT INTO 表名 (列1, 列2) VALUES ('值1', '值2'), ('值3', '值4');

-- 删除
DELETE FROM 表名 WHERE 条件;

-- 修改
UPDATE 表名 SET 列名 = 值 WHERE 条件;

-- 查询所有
SELECT * FROM 表名;

-- 查询指定列
SELECT 列名 FROM 表名;

-- 条件查询
SELECT * FROM 表名 WHERE 列名 = 值;
SELECT 列名 FROM 表名 WHERE 条件;
```

## 数据导出/导入

```bash
# 导出
mysqldump -u root -p 数据库名 > 数据库名.sql

# 导入
mysql -u root -p 数据库名 < 数据库.sql
```

## PyMySQL

### 连接

```python
import pymysql

connection = pymysql.connect(
    host='127.0.0.1',
    port=3306,
    user='root',
    passwd='密码',
    charset='utf8',
    db='数据库名'
)
```

### 创建游标

```python
cursor = connection.cursor(cursor=pymysql.cursors.DictCursor)
```

### 写操作（增删改）

```python
# 参数化（防 SQL 注入，只用 %s，pymysql 自动转换类型）
cursor.execute("INSERT INTO 表(列1, 列2) VALUES (%s, %s)", [值1, 值2])

# 字典传参
cursor.execute("INSERT INTO 表(列1, 列2) VALUES (%(k1)s, %(k2)s)", {'k1': 值1, 'k2': 值2})

# 提交
connection.commit()
```

> 不能用 `%(列)s` 字符串格式化的方式，必须用参数化查询防范 SQL 注入。

### 读操作（查询）

```python
cursor.execute("SELECT * FROM 表 WHERE 列 = %s", [值])

# 获取结果（返回列表套字典：[{列1: 值1, 列2: 值2}, ...]）
result = cursor.fetchall()
```

> `fetchall()` 游标移动后不可再次获取，第一次调用就要把结果赋值保存。

### 关闭

```python
cursor.close()
connection.close()
```
