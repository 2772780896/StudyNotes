# Requests 模块 & XPath

## Requests

```python
import requests
```

### GET 请求

```python
response = requests.get(url)

# 设置编码
response.encoding = 'utf-8'

# 获取页面源码
html = response.text

# 获取二进制数据（可写入文件后用其他编码查看）
binary = response.content
```

### POST 请求

```python
response = requests.post(url, data={'key': 'value'})

# JSON 响应直接转换
data = response.json()
```

> POST 数据格式可参考响应头末尾。

### 修改请求头

```python
response = requests.get(url, headers={'User-Agent': '...'})
```

### Session（保持会话）

```python
session = requests.Session()

# 登录
session.post('登录URL', data={'username': '...', 'password': '...'})

# 后续请求自动携带 Cookie
page = session.get('需要登录的URL')
```

## XPath

```python
from lxml import etree
```

### 解析 HTML

```python
tree = etree.HTML(html_string)
```

### 常用路径表达式

| 表达式 | 说明 | 示例 |
|------|------|------|
| `/html/body/div` | 绝对路径，从根开始 | 定位到 body 下的 div |
| `//div` | 从任意位置查找 div | 全局搜索所有 div |
| `//div/a/text()` | 获取标签文本内容 | 获取 a 标签的文字 |
| `//div/a/@href` | 获取属性值 | 获取 a 标签的 href |
| `//div[@class='box']` | 按属性筛选 | 定位 class 为 box 的 div |
| `./div` | 相对当前位置查找 | 在已获取标签基础上继续定位 |
| `/*` | 通配任意一级标签 | 跳过中间层级 |

### 使用示例

```python
tree = etree.HTML(html)

# 获取所有链接文字
texts = tree.xpath('//a/text()')

# 获取所有链接地址
hrefs = tree.xpath('//a/@href')

# 获取指定 class 的 div 下的 a 标签
links = tree.xpath("//div[@class='box']/a")

# 相对定位
div = tree.xpath('/html/body')[0]
inner = div.xpath('./div/a/text()')
```
