# 文件传输

## 前端上传

```html
<form method="post" enctype="multipart/form-data">
    <input type="file" name="file" />
</form>
```

> `enctype="multipart/form-data"` 必须设置，否则文件无法上传。

**通过 JS 获取文件：**

```javascript
input.addEventListener('change', function(e) {
    const file = e.target.files[0];  // 获取选择的文件
});
```

## 后端接收（Django）

文件数据存储在 `request.FILES`（不在 GET/POST 中），类型为 `MultiValueDict`：

```python
def upload(request):
    file = request.FILES.get('file')  # 获取上传的文件对象

    # 分块写入本地
    with open('保存路径/file.png', mode='wb') as f:
        for chunk in file.chunks():
            f.write(chunk)
```

## 文件保存策略

- 用户上传文件放在 `app/media/` 目录
- 数据库中只保存文件路径，不保存文件本身

```python
from openpyxl import load_workbook

wb = load_workbook('文件路径.xlsx')    # 打开 xlsx 文件
sheet = wb.worksheets[0]               # 获取第一个工作表

# 读取指定单元格
cell = sheet.cell(row=行数, column=列数)
value = cell.value

# 遍历行
for row in sheet.iter_rows(min_row=2):  # 从第 2 行开始（跳过表头）
    first_cell = row[0].value           # 该行第一列的值
```
