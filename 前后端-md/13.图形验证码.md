# 图形验证码（Pillow）

## 安装与导入

```bash
pip install pillow
```

```python
from PIL import Image, ImageDraw, ImageFont
```

## 生成图片

```python
# 创建空白图片（RGB 模式，尺寸，白色背景）
img = Image.new(mode='RGB', size=(宽, 高), color=(255, 255, 255))
```

## 创建画笔

```python
draw = ImageDraw.Draw(img, mode='RGB')
```

## 绘图操作

```python
# 画点
draw.point([x, y], fill=(R, G, B))

# 画线
draw.line((x1, y1, x2, y2), fill=(R, G, B))

# 画文字
draw.text([x, y], '文字', (R, G, B))

# 设置字体
font = ImageFont.truetype('字体路径', 字号)
draw.text([x, y], '文字', (R, G, B), font=font)
```

## 保存图片

```python
# 保存到文件
with open('file.png', mode='wb') as f:
    img.save(f, format='png')
```

## 返回图片给网页

```python
from io import BytesIO

stream = BytesIO()              # 内存中创建二进制文件
img.save(stream, 'png')

def img_show(request):
    return HttpResponse(stream.getvalue())
```
